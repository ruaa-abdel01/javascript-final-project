Images

Images can be added to HTML documents using the <img> tag. The src attribute specifies the path to the image, and the alt attribute provides alternative text for the image.

Example

<img src="path/to/image.jpg" alt="Description of image">

Attributes

  • src: Specifies the path to the image file.
  • alt: Provides alternative text for the image.
  • width: Sets the width of the image (in pixels or percentage).
  • height: Sets the height of the image (in pixels or percentage).

Example with Attributes

<img src="image.jpg" alt="A beautiful scenery" width="500" height="300">
