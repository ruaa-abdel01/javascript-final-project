How To Create A Paragraph

To create a paragraph, surround the words that make up the paragraph with the opening <p> and closing </p> tags.

By default, a browser will show each paragraph on a new line with a space between each paragraph. This is done automatically by the browser, so you do not need to add any extra spaces or line breaks.
