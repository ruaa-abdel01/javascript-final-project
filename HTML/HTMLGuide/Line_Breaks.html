Line Breaks & Horizontal Rules

Line Breaks

The Earth
gets one hundred tons heavier every day
due to falling space dust.

As you have already seen, the browser will automatically show each new paragraph or heading on a new line. But if you wanted to add a line break inside the middle of a paragraph, you can use the <br /> element.

Horizontal Rules

Venus is the only planet that rotates clockwise.

Jupiter is the largest planet in our solar system.

To create a break between themes—such as a change of topic in a book or a new section in a web page—you can add a horizontal rule between sections using the <hr /> element.
